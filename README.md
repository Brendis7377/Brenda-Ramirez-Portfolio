# Brenda Ramirez — Portfolio Website   

Deployment!
Live site:  
https://brendis7377.github.io/Brenda-Ramirez-Portfolio/

This is my personal portfolio website built with **HTML, CSS, and vanilla JavaScript**.  
I designed it to be clean, responsive, and accessible, with a focus on UI/UX fundamentals.

## ✨ Features
- **Responsive layout** using CSS Grid & Flexbox  
- **Light/Dark theme toggle** with `localStorage` + system preference support  
- **Sticky header** with smooth in-page navigation  
- **Accessible structure** using semantic HTML + ARIA attributes  
- **Project cards**, About section, and Contact section  

## :) Tech Used
- **HTML5** — semantic structure  
- **CSS3** — Grid, Flexbox, custom properties (CSS variables)  
- **JavaScript** — theme toggle + small interactive logic
